DEFINITION

HTML-Hypertext Markup Language
HEAD: metadata and links to resources
TITLE:name of the webpage
BODY:the information
HTML:the root of html document
Tags: code element used to define content and structure
< h1 > to < h6 >: used for headings, h1 being the largest, h6 being the smallest header
< p >: used for making paragraphs
< br >: used to separate texts
Attribute: information inside a tag that modifies its behavior
src: source for images,scripts etc
alt: alternative text for images
href: link destination
width and height: used to change the size of your image
Examples of attributes: src, height, width, alt
